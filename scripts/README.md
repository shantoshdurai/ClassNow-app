# Utility Scripts

This folder contains executable scripts for common development tasks.

## Scripts

- **new.sh** - Auto-generation or setup script for new features/components

## Running Scripts

On Linux/macOS:
```bash
./scripts/new.sh
```

On Windows:
```bash
bash scripts/new.sh
```

## Adding New Scripts

When adding new utility scripts, place them in this folder and add documentation here.
