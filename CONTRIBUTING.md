# Contributing to Class Now

Thank you for your interest in contributing to Class Now! We welcome contributions from the community.

## Getting Started

1.  **Fork the Repository**: Click the "Fork" button on the top right of the repository page.
2.  **Clone Your Fork**:
    ```bash
    git clone https://github.com/YOUR_USERNAME/Timewise-app.git
    cd flutter_firebase_test
    ```
3.  **Create a New Branch**:
    ```bash
    git checkout -b feature/your-feature-name
    ```

## Making Changes

*   **Code Style**: Please follow the standard Flutter/Dart style guide.
*   **Testing**: Ensure that your changes work as expected. Run existing tests if available.
*   **Documentation**: Update documentation if you add new features.

## Submitting a Pull Request

1.  **Push Your Changes**:
    ```bash
    git push origin feature/your-feature-name
    ```
2.  **Open a Pull Request**: Go to the original repository and click "Compare & pull request".
3.  **Describe Your Changes**: Provide a clear description of what you've done.

## Reporting Issues

If you find a bug or have a feature request, please open an issue in the repository.

Thank you for contributing!
